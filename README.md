# Urban Unknown
## About
3D godot game about urban exploring

### Main Idea
**3D game** with **co-op** and **proximity chat** taking inspiration from *lethal company*, *phasmophobia*.

### Ideas for maps and enemies
#### Maps
> * Sewers
> * Mental Hospital
> * Abandoned building
> * Chernobyl
> * Forest
> * Haunted House
> * Abbey
#### Enemies
> * Ghosts
> * Demons
> * Mutated Animals and people
> * Rat Man
> * Slender Man
> * Rake

### Assets
Fmod - proximity chat
Blender - models
Godot - engine
